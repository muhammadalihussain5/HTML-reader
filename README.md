# HTML code to Readable page/text Converter

A simple web app that converts raw HTML code into a clean, readable format.  
Paste your HTML, and the app will sanitize it (removing `<script>` and `<style>` tags) and display the translated content instantly.

---

## 🚀 Features
- **Live Conversion**: Paste HTML and see the readable output immediately.
- **Sanitization**: Removes unsafe `<script>` and `<style>` tags for safety.
- **Copy Output**: One-click button to copy the translated text.
- **Responsive Design**: Works smoothly on desktop and mobile.
- **Modern UI**: Styled with CSS variables, gradients, and shadows.

---

## 📂 Project Structure
Since CSS and JavaScript are embedded directly in the HTML file, the project consists of a single file:

- `index.html` → Contains HTML structure, embedded CSS styles, and JavaScript logic.
